# Plan 5.5: Internal Market Data API

## Objective
Build FastAPI endpoints to serve historical and latest market data to the frontend or other internal services.

## Context
- backend/market_data/api/router.py
- backend/market_data/storage/models.py
- backend/market_data/normalizers/schemas.py

## Tasks

<task type="auto" effort="medium">
  <name>Build Data Endpoints</name>
  <action>
    Update `backend/market_data/api/router.py` to include:
    - `/candles`: Fetch paginated recent candles by symbol and timeframe.
