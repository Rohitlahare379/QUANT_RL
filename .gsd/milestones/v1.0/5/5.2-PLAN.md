# Plan 5.2: Market Candle Database Schema

## Objective
Create a normalized database schema to store historical and live market candles (OHLCV) for the Aegis Market Data Layer.

## Context
- backend/database.py
- backend/main.py

## Tasks

<task type="auto" effort="medium">
  <name>Create MarketCandle Model</name>
  <action>
    Create `backend/market_data/storage/models.py`.
    Define the `MarketCandle` SQLAlchemy model with fields: id, symbol, timeframe, timestamp, open, high, low, close, volume, source, created_at.
    Add compound index and unique constraints for symbol + timeframe + timestamp.
  </action>
  <verify>grep "MarketCandle" backend/market_data/storage/models.py</verify>
  <done>Model successfully defined with indexes.</done>
</task>

  </action>

  <done>Model is hooked up to the startup table creation.</done>
</task>
